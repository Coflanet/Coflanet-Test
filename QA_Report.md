# Coflanet UI QA Report
> Figma 디자인 vs Flutter 구현 비교
> 작성일: 2026-03-05

---

## 1. Splash Screen

### Figma 디자인
- 배경: 단색 보라색 (바이올렛)
- 중앙: Coflanet 로고 (구름 형태) + "Coflanet" 텍스트
- 상태바: 밝은 아이콘

### Flutter 구현
- 배경: **그라디언트** (좌상→우하) `#6541F2` → `#4F29E5`
- 로고: `logo_splash.png` (200x200px) 중앙 배치
- 상태바: light 아이콘, 투명 배경

### QA 결과
| 항목 | 상태 | 비고 |
|------|------|------|
| 배경색 | ⚠️ 차이 | Figma는 단색, 코드는 그라디언트. 의도된 것인지 확인 필요 |
| 로고 | ✅ 일치 | 중앙 배치, 적절한 크기 |
| 상태바 | ✅ 일치 | 밝은 아이콘 스타일 |
| 전환 로직 | ✅ | 2초 후 자동 전환 |

## 2. 로그인 (Sign In)

### Figma 디자인
- 배경: 흰색
- 제목: "로그인하고\n내 취향을 찾아볼까요?" (중앙 정렬)
- 버튼 3개: 카카오(노란), 네이버(초록), Apple(검정)
- 하단: "회원가입 | 게스트로 로그인"

### Flutter 구현
- 배경: `backgroundNormalNormal` (#FFFFFF)
- 제목: heading1Bold, `labelNormal` (#171717), lineHeight 1.4
- 버튼: 52px 높이, 12px 간격, borderRadius 12px
  - 카카오: `#FEE500`, "카카오로 3초만에 시작하기"
  - 네이버: `#03C75A`, "네이버로 로그인"
  - Apple: `#000000`, "Apple로 로그인" (iOS only)
- 하단: "이메일 로그인 | 게스트로 로그인"

### QA 결과
| 항목 | 상태 | 비고 |
|------|------|------|
| 배경 | ✅ 일치 | 흰색 |
| 제목 텍스트 | ✅ 일치 | |
| 카카오 버튼 색상 | ✅ 일치 | #FEE500 (공식 카카오 색상) |
| 네이버 버튼 색상 | ✅ 일치 | #03C75A (공식 네이버 색상) |
| Apple 버튼 | ⚠️ 주의 | iOS 전용, Android에서는 미표시 |
| 하단 링크 | ⚠️ 차이 | Figma: "회원가입", 코드: "이메일 로그인" (텍스트 불일치) |
| 버튼 높이 | ✅ | 52px |
| 버튼 둥글기 | ✅ | 12px border-radius |

## 3. 온보딩 - 이름 입력 (Profile Setup)

### Figma 디자인
- 상단 프로그레스 바 (50%)
- 제목: "반가워요! 👋\n이름을 입력해주세요"
- 입력 필드: "이름" 힌트 텍스트
- 하단: "확인" 버튼

### Flutter 구현
- 프로그레스: SurveyProgressIndicator(0.5), 4px 높이, violet 색상
- 제목: title2Bold (28px, w700), #171717
- 입력 필드: TextFieldSize.lg (56px), 힌트 "이름 또는 닉네임"
- 버튼: "시작하기", 52px, primaryNormal, 이름 2글자 이상 시 활성화

### QA 결과
| 항목 | 상태 | 비고 |
|------|------|------|
| 프로그레스 바 | ✅ 일치 | |
| 제목 텍스트 | ✅ 일치 | 이모지 포함 |
| 입력 힌트 | ⚠️ 차이 | Figma: "이름", 코드: "이름 또는 닉네임" |
| 버튼 텍스트 | ❌ 불일치 | Figma: "확인", 코드: "시작하기" |

## 4. 온보딩 - 가입 이유 (Survey Reason)

### Figma 디자인
- 제목: "커플래닛을 찾게 된\n이유를 알려주세요."
- 부제: "중복 선택 가능해요."
- 5개 체크박스 옵션
- 선택 시: 보라색 테두리 + 보라색 텍스트
- 하단: "완료" 버튼

### Flutter 구현
- 제목: heading1Bold (22px, w700)
- 부제: body1NormalRegular (16px, w400), labelAlternative 색상
- 체크박스: AnimatedContainer, 12px 둥글기
  - 미선택: componentFillNormal 배경, transparent 테두리
  - 선택: white 배경, 2px primaryNormal 테두리
- 아이템 간격: 12px
- 버튼: "완료", 52px, 1개 이상 선택 시 활성화

### QA 결과
| 항목 | 상태 | 비고 |
|------|------|------|
| 레이아웃 | ✅ 일치 | |
| 선택 스타일 | ✅ 일치 | 보라 테두리 + 보라 텍스트 |
| 미선택 스타일 | ✅ 일치 | 회색 배경 |
| 간격 | ✅ 일치 | 12px |
| 버튼 | ✅ 일치 | "완료" |
| 애니메이션 | ✅ 추가 | 200ms 전환 (Figma에 없지만 UX 개선) |

## 7. 설문조사 - 질문 화면

### Figma 디자인
- 상단: 뒤로가기 + "커피 추출 방식 선택" + 아이콘
- 프로그레스 바
- "어떤 기구로\n커피를 마시나요?"
- "중복 선택 가능해요."
- 5개 이미지 카드 (2열 그리드)
- 선택 시: 보라색 테두리
- "잘 모르겠어요" 링크
- "선택했어요" 버튼

### Flutter 구현
- AppBar: 뒤로가기 + 센터 타이틀 + X 버튼
- 다양한 질문 타입 지원:
  - 체크박스 (텍스트)
  - 아이콘 체크박스
  - 3단계 평가 (👎😐👍) - 빨/회/파
  - 2단계 평가 (👎👍)
  - 이미지 그리드 (2열)
  - 복합 평가
- 자동 넘김: 평가 질문은 선택 시 자동 다음 질문

### QA 결과
| 항목 | 상태 | 비고 |
|------|------|------|
| 이미지 그리드 | ✅ 일치 | 2열, 보라 테두리 선택 |
| 프로그레스 바 | ✅ 일치 | 섹션별 진행률 |
| 다중 선택 | ✅ 일치 | |
| "잘 모르겠어요" 옵션 | ✅ 일치 | |
| AppBar 우측 | ⚠️ 차이 | Figma: 아이콘 버튼, 코드: X(닫기) 버튼 |
| 자동 넘김 | ✅ 추가 | 평가 질문 자동 전환 (UX 개선) |

## 8. 설문 결과 화면

### Figma 디자인
- 상단: "나의 커피 취향" 타이틀
- 보라색 배너: "[이름]님은\n진하고 깊은 풍미를\n즐기시네요! ☕"
- 취향 4칸 그리드: 산미/바디감/단맛/쓴맛 (좋음/보통/싫음)
- 4가지 향 설명 카드
- "추천 원두" 섹션: 원두 카드 목록 (일치율, 맛 그래프, 플레이버 태그)
- 하단: "총 X개 원두 목록 추가" 버튼

### Flutter 구현
- CustomScrollView + Sliver 구조
- 보라색 그라디언트 배너: Violet 60→50, 24px padding, xlBorder
- 4칸 그리드: 이모지 표시 (👍/😐/👎, 점수 기반)
- 향 설명: 보라 그라디언트 원형 아이콘 (48x48)
- 추천 카드: pill shape (40px radius), 보라 테두리, 체크박스, 맛 바 차트
- 하단 CTA: 선택된 원두 수 표시

|------|------|------|

| 취향 그리드 | ✅ 일치 | 4칸, 이모지 매핑 |
| 향 카드 | ✅ 일치 | |

## 9. 원두 목록 (Coffee List)

### Figma 디자인
- 상단: "원두 목록" + "편집" 버튼
- 원두 카드: 접기/펼치기 (아코디언)
  - 펼친 상태: 맛 프로필 바 (산미/바디감/단맛/쓴맛/밸런스)
  - 플레이버 태그 (chip)
  - "원두 상세" / "레시피 실행" 버튼
- + 버튼 (원두 추가)
- 하단 탭바: 원두 | 추출 목록 | 시음 기록 | My 행성

### Flutter 구현
- 검정 배경 쉘 + 라운드 흰색 콘텐츠 (40px 상단 둥글기)
- 아코디언 카드 구현
- 맛 프로필 바 차트
- 플레이버 태그
- 편집 모드: 탭바 숨김, 보라 "완료" 버튼
- 하단 탭바: glass effect pill (328x64), 4탭

### QA 결과
| 항목 | 상태 | 비고 |
|------|------|------|
| 아코디언 카드 | ✅ 일치 | 접기/펼치기 동작 |

| 플레이버 태그 | ✅ 일치 | chip 스타일 |
| 탭바 디자인 | ⚠️ 차이 | Figma: 단순 아이콘+텍스트, 코드: glass effect pill 컨테이너 (더 세련됨) |
| 탭바 아이콘 | ⚠️ 확인 필요 | Figma: 커스텀 아이콘, 코드: Material 아이콘 |
| 편집 모드 | ✅ | |
